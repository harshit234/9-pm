Q-1 Shallow Copy vs Deep Copy
Shallow Copy-Copies the reference of nested objects, not the actual object.
Example:

import copy
a = [[1,2],[3,4]]
b = copy.copy(a)

b[0][0] = 99

print(a)

Output

[[99,2],[3,4]]

Both lists change because nested list reference is shared.

Deep Copy
Creates completely independent copies of all nested objects.

import copy

a = [[1,2],[3,4]]
b = copy.deepcopy(a)

b[0][0] = 99

print(a)

Output

[[1,2],[3,4]]

Now original list remains unchanged.

When to Use Deep Copy

Use deep copy when working with:

nested lists

dictionaries inside lists

complex data structures

Q2 — Rotate List
def rotate_list(lst, k):

    n = len(lst)
    k = k % n

    return lst[-k:] + lst[:-k]


print(rotate_list([1,2,3,4,5], 2))

Output - [4,5,1,2,3]

Q3 — Debug Problem
Given Code
nums = [1,2,3,4,5,6,7,8]

for num in nums:
    if num % 2 == 0:
        nums.remove(num)

print(nums)
Bug Reason

When removing elements while iterating, the list index shifts.

Example

[2,4,6,8]

Iteration skips elements because positions change.

Correct Solution - Use list comprehension.

nums = [1,2,3,4,5,6,7,8]
nums = [n for n in nums if n % 2 != 0]
print(nums)

Output - [1,3,5,7]
